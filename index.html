Acerca de Api Turismo

Esta API tiene como propósito brindar una herramienta educativa para futuros desarrolladores y estudiantes.
Através de ella podrán :

  • Explorar la riqueza cultural y gastronómica del Ecuador.
  • Aprender a consumir datos de una API sobre lugares turísticos y platos típicos del Ecuador.
  • Este recurso te permitira mejorar el conocimiento en programación y diseño de servicios web, mientras se conoce más sobre la diversidad de Ecuador.

Lugares Turisticos

Descripción del endpoint /turismo:

  • Función: Devuelve todos los lugares turísticos disponibles.
  • Respuesta: Un JSON de objetos que contienen información de todos los lugares turísticos.
  • Método HTTP: GET

Descripción del endpoint /turismo/turismoid:
(turismoid hace referencia a un lugar en específico)

  • Función: Devuelve los detalles de un lugar turístico específico.
  • Respuesta: Un JSON de objetos que contienen información de un lugar turístico.
  • Método HTTP: GET

Gastronomia

Descripción del endpoint /grastronomia:

  • Función: Devuelve datos sobre la gastronomía más relevante del país.
  • Respuesta: Un JSON de objetos que contienen información sobre los platos típicos más destacados.
  • Método HTTP: GET

Provincias

Descripción del endpoint /provincias:

  • Función: Devuelve todos las provincias del país.
  • Respuesta: Un JSON de objetos que contienen información de todas las provincias y regiones.
  • Método HTTP: GET
